Introdução à Tag <img>

A tag <imggt em HTML é usada para incorporar imagens em uma página web. Ela é um elemento vazio, ou seja não possui uma tag de fechamento e contém apenas atributos que definem o centeúdo visual.

Atributos Principais:

  • src (source): define o caminho da imagem que será exibida. pode ser uma URL externa ou um caminho relativo dentro do servidor.

    Exemplo 1:

    <img src="./image.webp" style="width: 100%;">

  • alt (aternantive text): fornece um texto alternativo para ser exibido caso a imagem não possa ser carregada.Ele também melhora a acessibilidade para leitores de tela.

Tipos Comuns de Imagens

Os formatos de imagens mais ultilizados na web são:

  • (.jpg ou .jpeg): Formato utilizado amplamente para fotografias e imagens ccom muitos detalhes e core. Possui compressão com perde de qualidade (lossy).
  • (.png): Usado para imagens com fundo transparente ou que necessitam de alta qualidade, sem perda de detalhes. Utiliza compressão sem perda (lossless).
  • GIF (.gif): Formato ideal para imagens animdas e pequenos ícones. Suporta transparência, mas tem um número limitido de cores.
